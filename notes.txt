MongoDB is a NoSQL database that stores data in a flexible, JSON-like format called BSON (Binary JSON). It stores data in collections, each collection contains documents which is individual data entities.

Advantages of MongoDB:
   1. Performance: MongoDB is designed for high performance and scalability.
   2. Flexibility: MongoDB is easy to make structural changes
   3. Scalability: MongoDB can support high request rate

We need to install `mongoose` package with the following command:
   `npm i mongoose`
